# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-03-16

### Added
- **15 specialized agents** covering the full software development lifecycle
- **Strategy:** product-strategist, project-manager
- **Design:** ui-ux-designer
- **Architecture:** architect, database-expert
- **Development:** flutter-dev, backend-dev, swift-dev
- **Quality:** code-reviewer, test-engineer, debugger
- **Security:** security-auditor
- **Operations:** devops-engineer, performance-optimizer
- **Growth:** marketing-advisor
- Workflow chains: agents can work together in defined sequences
- Standardized output format across all agents
- `install.sh` for one-command setup
- `uninstall.sh` for clean removal
- Comprehensive README with usage examples
- ARCHITECTURE.md explaining design decisions
- CONTRIBUTING.md for contributors
- GitHub Actions CI for validation
